# Calculator Program in C

This repository contains a simple calculator program written in C language.

## Description
This program performs basic arithmetic operations such as
Addition, Subtraction, Multiplication and Division.

The user selects the operation from menu
and enters two numbers to get the result.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Menu driven program
